// 다국어 지원 사전 파일

// #기본 설정
//	영어 = 한국어
//	왼쪽이 영어 오른쪽이 한국어로 지정한다.
//
// #주석 설정
//	슬래시 두개로 주석처리 가능함
//
// #그룹 설정
//	[group=그룹이름] 으로 그룹 설정 가능함
//	씬에서의 그룹설정은 LocalizationGroup 컴포넌트를 
//	추가해서 지정 가능 함.
//	팝업창이 열릴때 활성화 되는 오브젝트에 이 컴포넌트를
//	추가해야 하위의 텍스트 오브젝트에 적용된다.
//
// #텍스트 ID 설정
//	[ID_이름] 으로 텍스트 ID를 지정 가능함
//	주로 코드에 박혀 있는 텍스트앞에 [ID]를 지정 하고
//	ID_이름 으로 찾아 글자를 치환한다.
//
